# AI-Powered Job Marketplace (Flutter)

A **production-style AI-powered Job Marketplace** built with Flutter and Riverpod, featuring real-time chat, paid AI job analysis, CV upload, bookmarks, and a scalable backend architecture.

This project was developed **for portfolio purposes** to demonstrate advanced Flutter development, clean architecture, and real-world SaaS features.
